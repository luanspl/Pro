Git Commands:

git init -> Iniciar um repositório vazio
git status -> Visualizar o Status do repositório
git add 'nome do arquivo'
git commit -m 'msgm'
git log
git remote add 'origin' 'site'
.gitignore -> file with file names to ignore

git add file
git add -A -> all files
git reset file -> remove file from Stage area
git log -> see the commits made
